# JS Starter Project
Starter project for JS based apps/sites.

## Development
- Webpack
  - Babel loader
    - React 
    - ES2015
  - ESLint preloader

## Scripts
- `npm run build` - Runs webpack build (with eslint as a prestart process)
- `npm run build-doc` - Creates JSDocs
- `npm run test` - Runs mocha tests
